## Local Setup

Clone the repository with:

```bash
git clone https://github.com/zaid-commits/Discord-Invite.git
```

Switch to the main directory:

```bash
cd Discord-Invite
```

### Install & run

Install dependencies with:

```bash
npm install
```

Once it's done start up a local server with:

```bash
npm run dev
```
